# Unit Converter 📏⚖️🕒

This is a simple, command-line Python program designed to perform various **Length**, **Mass**, and **Time** unit conversions. It serves as a handy tool for quick calculations between metric and imperial units for common measurements.

## ✨ Features

The program offers the following conversion categories and specific conversions:

### 📐 Length Conversions
* **Kilometers (km)** to **Meters (m)**
* **Meters (m)** to **Centimeters (cm)**
* **Meters (m)** to **Feet (ft)** (using an approximation)
* **Inches (in)** to **Centimeters (cm)**

---

### ⚖️ Mass Conversions
* **Kilograms (kg)** to **Grams (g)**
* **Grams (g)** to **Milligrams (mg)**
* **Kilograms (kg)** to **Pounds (lb)** (using an approximation)

---

### 🕒 Time Conversions
* **Hour (h)** to **Minute (min)**
* **Minute (min)** to **Second (s)**
* **Second (s)** to **Minute (min)**
* **Minute (min)** to **Hour (h)**
* **Second (s)** to **Hour (h)**
* **Hour (h)** to **Second (s)**

---

## 🚀 Getting Started

### Prerequisites

You need to have **Python 3** installed on your system to run this program.

### Installation

No special installation is required. Simply save the provided code into a file named, for example, `unit_converter.py`.

### How to Run

1.  Open your terminal or command prompt.
2.  Navigate to the directory where you saved `unit_converter.py`.
3.  Run the program using the Python interpreter:

    ```bash
    python unit_converter.py
    ```

## 🖥️ How to Use

The program runs in a continuous loop until you choose to exit, prompting you for input at each step:

1.  **Main Menu:** Select the category you want to convert: `1` for Length, `2` for Mass, or `3` for Time.
2.  **Sub-Menu:** Select the specific conversion you wish to perform within that category (e.g., Kilometers to Meters).
3.  **Input Value:** Enter the numerical value you want to convert. The program handles both integers and decimal numbers.
4.  **Result:** The converted result will be displayed, typically rounded to two decimal places.
5.  **Rerun/Exit:** After each conversion, you'll be asked to type `'rerun'` to perform another conversion or `'exit'` to close the program.
